# 🐦 Exclusor de Tweets
Script para deletar posts (tweets) no X, antigo Twitter.

## Tecnologias
- Python
- Selenium
- Numpy

## Instalação
```bash
pip install -r requirements.txt
```

## Uso
Substitua as variáveis presentes no script:

- `user`: seu usuário

- `password`: sua senha

- `URL_DA_PAGINA`: URL do seu perfil

## Execução
```bash
py file.py
```
